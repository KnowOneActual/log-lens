# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-23

### Added
- **Core Parser**: Implemented regex-based log level detection.
- **IP Tracking**: Added IPv4 extraction and ranking of top active IPs.
- **JSON Export**: Added `--export` flag to save analysis as JSON.
- **CLI**: Built `argparse` interface for file handling.
- **Tests**: Added unit tests for `LogParser` logic.
- **Tag**: 1.0.0
